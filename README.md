# Keybinds

| Key         | Action        |
|-------------|---------------|
| `q`         | Quit          |
| `j` / `↓`   | Scroll Down   |
| `k` / `↑`   | Scroll Up     |
| `m`         | Add or delete Bookmark|
| `b`         | Bookmark Menu |
| `→` / `l`   | Switch Fcons Left  |
| `←` / `h`   | Switch Fcons Right |
